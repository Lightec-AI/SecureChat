# SecureChat Mobile Wrapper

Capacitor-based mobile shell for:

- Android
- iPhone
- iPad (via iOS target)

## Setup

From repo root:

- `npm run mobile:postinstall`
- `npm run mobile:sync`

Open native projects:

- `npm run mobile:android`
- `npm run mobile:ios`

By default, the wrapper points to `http://localhost:3080`.
Set `SECURECHAT_MOBILE_URL` to target another environment.
